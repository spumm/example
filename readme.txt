					Общие сведения по программе ООО "БФЗ" DataCollector.

Программа предназначена для сбора данных АСУТП с оборудования ООО "БЗФ" и записи их в единую базу данных.
База данных расположена по адресу db.bzf.asu. Установлен PostgresSQL 11 на Windows 10.
Предполагается, что эта программа будет выполнятся, непосредственно на сервере базы данных.

На момент написания программы был следующий перечень оборудования, с которого требуется собирать данные:
РТП-1 - данные берутся с FTP сервера, по адресу ftp://rtp1-server.bzf.asu. Скачиваются двоичные файлы, и построчно разбираются.
График выполнения этой функции, соответствует записи значений программой на самом сервере RTP-1.
В периоды 00:00 до 01:00, С 8:00 до 9:00, по одному разу, собирает значения сразу за 8 часов. С 9:00 до 16:00 ежечасно.
РТП-2 - данные берутся с MySQL сервера, по адресу rtp2-arm.bzf.asu. Данные можно собирать ежечасно.
РТП-3 - на реконструкции
РТП-4 - имеется SQL SERVER по адресу rtp4-server.bzf.asu, с минутными и часовыми значениями.
Данные с трендами хранятся в двоичных файлах, доступ к которым организован через программу HistData.exe, посредством DDE объектов.
ДО - данные по навескам берутся с MySQL 3.23 сервера, по адресу batch-arm.bzf.asu
ДО Графики - данные по технологическим параметрам ДО берутся с MySQL 3.23 сервера, по адресу batch-arm.bzf.asu
ГОУ Графики - берутся с 3 MySQL серверов, расположенных по адресам: goy1-arm.bzf.asu, goy2-arm.bzf.asu, goy4-arm.bzf.asu

Демонический режим - выполняет сбор данных согласно графику установленному в main.py, методе daemon()

Внимание: для работы программы требуется наличие папки с правами записи по адресу \\db.bzf.asu\\import\
